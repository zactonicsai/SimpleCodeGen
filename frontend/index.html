Chroma + Ollama Code Generator

Simple HTML forms that send your prompt to a local FastAPI backend using ChromaDB + Ollama to generate custom code.

General Code Generator

Describe what you want (e.g., “React form with Tailwind”), choose a language, and get generated code.

More templates (todo)

You can duplicate code-generator.html and create specialized forms for APIs, UI components, tests, etc.

Backend expected at http://localhost:8000.
